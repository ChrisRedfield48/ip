# NIP.SYS v3.0

Personal portfolio dashboard of **Nikita Ip** — web developer based in Belgrade, Serbia.

## Stack

- HTML / CSS / Vanilla JS
- Google Fonts (Bebas Neue, DM Mono, Inter Tight)
- Open-Meteo API (live weather)

## Features

- **Bento grid** dashboard with animated cards
- **Live weather** for Belgrade
- **Terminal** with custom commands (`help`, `skills`, `projects`, `matrix`, `quote`, `advice`...)
- **Session uptime** + countdown timer to transfer deadline (March 2029)
- **Now Playing** — fake music player with equalizer animation
- **Location card** — animated map marker for Belgrade
- **Guestbook** — persistent messages via localStorage
- Custom cursor, grain overlay, scanlines, glitch effects
- Low-perf mode for weak devices / reduced motion

## Run locally

```bash
git clone https://github.com/ChrisRedfield48/ip.git
cd ip
open index.html
```

No build step. No dependencies. Just open the file.
