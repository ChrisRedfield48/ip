# SYSTEM.IP — Никита Ип

> Персональный дашборд-портфолио в стиле OS/терминала

**[→ LIVE DEMO](https://chrisredfield48.github.io/ip/)**

---

## Стек

- HTML / CSS / JavaScript
- Open-Meteo API (погода)
- Google Fonts (Space Mono, Syne)

## Фичи

- Preloader с анимацией загрузки
- Интерактивный терминал с командами (`help`, `skills`, `projects`, `matrix` и др.)
- Живой счётчик uptime и обратный отсчёт до трансфера
- Погода в реальном времени (Белград)
- Модуль симуляции обучения нейросети
- Переключение режимов (фокус / гринд / отдых / зал)
- Адаптивная верстка

## Запуск локально

```bash
git clone https://github.com/ChrisRedfield48/ip.git
cd ip
open index.html
```
